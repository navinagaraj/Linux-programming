STEP 1: Name your executable as my_ls

STEP 2: Executing my_ls without parameters has to take the listing directory as current directory.

STEP 3: Executing my_ls with one argument has to check wether the passed argument is a directory.

STEP 4: Executing my_ls with one argument has to read the directory contents and display the list of files and directory mentioned in the directory mentioned in the argument.
